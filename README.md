#  React Custom Hook - use-Fetch-hook Project

## - Overview

This project demonstrates how to create and use a custom React hook called `useFetch` to simplify API data fetching.

Instead of writing fetch logic repeatedly in every component, this hook handles:
- API requests
- Loading state
- Error handling

The project fetches product data from a public API and displays it in a clean UI.



## Features

- Custom Hook (`useFetch`)
- API data fetching using `fetch`
- Loading indicator
- Error handling
- Reusable logic
- Responsive product UI



##  Custom Hook: useFetch

The `useFetch` hook is designed to:
- Accept a URL as input
- Fetch data from the API
- Manage:
  - `data`
  - `loading`
  - `error`

This improves code reusability and keeps components clean.
